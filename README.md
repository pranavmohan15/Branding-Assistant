# Branding Assistant

Branding Assistant is a simple Streamlit application that helps generate brand names, slogans, and logos using AI.

The user enters a business idea, and the app suggests creative brand names. After selecting a name, the app generates catchy slogans and can also create an AI-generated logo based on the brand.

## Features

- Generate creative brand names
- Create catchy slogans for a selected brand
- Generate AI logos based on brand style
- Simple interactive UI built with Streamlit

## Tech Stack

- Python
- Streamlit
- OpenAI API
- python-dotenv
